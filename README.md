# 🛒 Blinkit Sales Dashboard

## 📌 Overview
This is an interactive Power BI dashboard built using the Blinkit Grocery dataset. The dashboard provides insights into sales performance, outlet analysis, customer ratings, and product categories.

## 📊 Dashboard Features
- 💰 Total Sales
- 📈 Average Sales
- 📦 Number of Items
- ⭐ Average Rating
- 🏪 Outlet Performance
- 📍 Outlet Location Analysis
- 🥧 Item Type Analysis
- 🎛 Interactive Slicers

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
